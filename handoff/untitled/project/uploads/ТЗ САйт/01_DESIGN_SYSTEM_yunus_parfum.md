# YUNUS PARFUM — Дизайн-система

Премиальный лендинг ателье селективной парфюмерии на разлив (Санкт-Петербург).
Документ передаётся в **Claude Design**. Основан на референсе (тёплый кремовый премиум-стиль), но адаптирован под нишевую восточную парфюмерию: масла, амбра, золото.

---

## 0. Бренд в одном предложении

> «Подберите аромат, выберите формат и объём — мы соберём ваши духи вручную».

Тон: дорого, спокойно, без крика. Ощущение бутика селективной парфюмерии, а не масс-маркета. Эффект «вау» создаётся воздухом (много пустого пространства), типографикой с высоким контрастом, золотым акцентом и макро-фотографией масел.

---

## 1. Цветовая палитра

Тёплая, «амбровая». 4 базовых + 2 акцента. Все значения — в HEX и в виде CSS-переменных.

| Роль | Название | HEX | Применение |
|---|---|---|---|
| Фон базовый | `--bg-cream` | `#F4ECDD` | Основной фон страницы |
| Фон тёплый-светлый | `--bg-ivory` | `#FBF5EA` | Карточки, секции-чередование |
| Поверхность тёмная | `--espresso` | `#2C1B10` | Футер, тёмные секции, hero-overlay |
| Текст основной | `--ink` | `#241710` | Заголовки и основной текст |
| Текст приглушённый | `--ink-muted` | `#6E5C4C` | Подписи, описания, плейсхолдеры |
| Акцент кофейный | `--mocha` | `#7A4F2C` | Кнопки primary, ссылки, иконки |
| Акцент золото | `--gold` | `#C2A05A` | Цены, разделители, «премиум»-детали, hover |
| Линии/границы | `--line` | `#E2D4BF` | Бордеры карточек, hairline-разделители |

Градиент для «золотого» акцента (бейджи, обводки):
```
--grad-gold: linear-gradient(135deg, #E4C988 0%, #C2A05A 45%, #9C7B3A 100%);
```

Тёмный overlay для hero поверх фото:
```
--overlay: linear-gradient(180deg, rgba(44,27,16,.10) 0%, rgba(44,27,16,.55) 100%);
```

Правило контраста: золото `--gold` НЕ используется для длинного текста (только цены, цифры, тонкие линии, иконки). Длинный текст — только `--ink` / `--ink-muted` на светлом и `#F4ECDD` на тёмном.

## 2. Типографика

Высокий контраст «дисплейный серив + спокойный книжный серив + утилитарный гротеск для цифр и UI». Все три шрифта имеют полную кириллицу и есть в Google Fonts.

| Роль | Шрифт | Начертания | Где |
|---|---|---|---|
| Дисплей / H1–H2 / логотип | **Playfair Display** | 600, 700, + italic | Hero-заголовок, названия секций, логотип |
| Книжный текст / H3–H4 / описания | **Lora** | 400, 500, 600 | Подзаголовки, абзацы, карточки |
| Утилита / UI / цены / лейблы | **Inter** | 400, 500, 600 | Кнопки, навигация, цены, бейджи, формы, надписи капсом |

> Playfair Display повторяет высокий контраст шрифта из референса. Если потребуется ещё более «кутюрный» вид — допустима замена дисплея на **Cormorant Garamond** (тоже кириллица), но по умолчанию используем Playfair Display.

### Подключение
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:ital,wght@0,600;0,700;1,600&family=Lora:wght@400;500;600&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
```

### Тип-шкала (desktop → mobile)
| Токен | Размер / интерлиньяж | Шрифт | Трекинг |
|---|---|---|---|
| `display` (H1 hero) | 72px / 1.05 → 40px | Playfair 700 | -0.5px |
| `h2` | 48px / 1.12 → 30px | Playfair 700 | -0.3px |
| `h3` | 30px / 1.2 → 22px | Playfair 600 | 0 |
| `h4` | 22px / 1.3 → 19px | Lora 600 | 0 |
| `body-lg` | 19px / 1.6 → 17px | Lora 400 | 0 |
| `body` | 16px / 1.65 | Lora 400 | 0 |
| `price` | 20px / 1.2 | Inter 600 | 0 |
| `eyebrow` (надзаголовок) | 13px / 1.4 | Inter 600 | +2.5px, UPPERCASE |
| `caption` | 13px / 1.5 | Inter 400 | +0.3px |
| `button` | 15px / 1 | Inter 600 | +0.5px |

«Эйбро» (eyebrow) — тонкая золотая надпись капсом над каждым H2 (напр. `СЕЛЕКТИВНАЯ ПАРФЮМЕРИЯ`). Это фирменный приём, повторяющий ритм референса.

## 3. Сетка, отступы, радиусы

- Контейнер: `max-width: 1240px`, боковые поля 24px (mobile) / 40px (desktop).
- Сетка каталога: 3 колонки (desktop) → 2 (tablet) → 1 (mobile), gap 24px.
- Вертикальный ритм секций: padding `120px` сверху/снизу (desktop), `72px` (mobile).
- Радиусы: кнопки — **pill** (`999px`, как в референсе); карточки — `18px`; изображения в карточках — `14px`; модалка/конфигуратор — `24px`.
- Шкала отступов (8-pt): 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 120.

### Тени (тёплые, мягкие, без серого)
```
--shadow-sm: 0 2px 8px rgba(60,40,20,.06);
--shadow-md: 0 10px 30px rgba(60,40,20,.10);
--shadow-lg: 0 24px 60px rgba(60,40,20,.14);
```

## 4. Компоненты

### Кнопки
- **Primary**: фон `--espresso` (или `--mocha`), текст `#FBF5EA`, pill, padding `16px 30px`, hover — затемнение + лёгкий подъём (`translateY(-1px)` + `--shadow-md`).
- **Secondary**: прозрачный фон, обводка `1.5px --mocha`, текст `--ink`, pill; hover — фон `--mocha`, текст кремовый.
- **Gold / премиум-CTA**: фон `--grad-gold`, текст `--espresso`, лёгкое внутреннее свечение; используется только для главного CTA hero и кнопки «Подобрать аромат».

### Карточка аромата
```
┌───────────────────────────────┐
│  [фото масла/флакона 4:5]      │
│                                │
│  EYEBROW: МУЖСКОЙ              │  ← мелкий золотой лейбл-категория
│  Название аромата (Lora 600)   │
│  короткое описание ноты        │  ← 1 строка, ink-muted
│                                │
│  от 100 ₽/мл        [Собрать →]│  ← цена золотом + кнопка
└───────────────────────────────┘
```
Hover: подъём `translateY(-4px)`, `--shadow-lg`, золотая hairline-обводка `--gold`.
Бейдж «ХИТ» / «ТОП» — для популярных (Baccarat 540, Dior Sauvage, Molecula 02 и т.п.).

### Бейджи объёма (внутри конфигуратора)
Pill-таблетки: `3 · 5 · 10 · 15 · 30 · 50 · 100 мл`. Активная — фон `--espresso`, текст кремовый. Неактивные — обводка `--line`.

### Фильтры каталога
Сегмент-контрол (3 таблетки): `Мужские · Женские · Универсальные`. Активная — золотая подложка `--grad-gold` или `--espresso`. Плюс поле поиска по названию.

### Хедер
Прозрачный поверх hero, при скролле — фон `--bg-ivory` + `--shadow-sm`. Слева логотип `YUNUS PARFUM`, справа навигация (Inter) + золотой CTA «Подобрать аромат».

## 5. Фотостиль

- Тёплый свет, кремовый/деревянный фон, кофейные зёрна, янтарные масла на просвет — как на реальных фото точки.
- Макро капли масла, флаконы с золотыми крышками, арабские декоративные флаконы.
- Hero — крупный флакон или капля масла с золотым бликом на кремовом фоне с лёгким `--overlay`.
- Не использовать холодные/синие стоковые фоны. Любая фотография «греется» к амбре.

---

## 6. Сигнатурный элемент (то, что запомнится)

**«Живой расчёт аромата» (Aroma Configurator).** При выборе объёма на ползунке/таблетках флакон на экране визуально «наполняется» золотистым маслом, а цена пересчитывается в реальном времени. Это прямое отражение сути бизнеса: клиент собирает свои духи. Один яркий интерактив — всё остальное вокруг спокойное и дисциплинированное.

---

## 7. Микро-анимации (сдержанно)

- Появление секций при скролле: fade + `translateY(16px)`, 500ms ease-out, по одному разу.
- Hover карточек/кнопок: 180ms.
- Наполнение флакона в конфигураторе: 400ms ease.
- Обязательно уважать `prefers-reduced-motion` — отключать сдвиги.

---

## 8. Доступность / качество

- Контраст текста к фону ≥ 4.5:1 (золото — только для крупных цифр/декора).
- Видимый фокус на интерактивах (золотая обводка 2px).
- Полная адаптивность до 360px.
- Кликабельные зоны ≥ 44px.
